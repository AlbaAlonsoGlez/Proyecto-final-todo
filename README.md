# Whisker Notes ₍^. .^₎⟆

> *Una elegante y minimalista aplicación de tareas creada con Spring Boot y un frontend moderno.*

---

## ♡ Sobre el proyecto

**Whisker Notes** es una aplicación web de gestión de tareas desarrollada con:

- Java Spring Boot
- HTML + CSS + Bootstrap
- JavaScript
- Arquitectura MVC

La aplicación permite crear, editar y organizar tareas mediante una interfaz limpia, responsive y agradable visualmente.

---

## ♡ Features

- Login y register
- CRUD tareas
- CRUD categorías
- CRUD etiquetas
- Diferenciación de tipos de usuarios
- Interfaz clara, moderna y responsive

---

## ♡ Tecnologías utilizadas

| Backend | Frontend | Extras |
|---|---|---|
| Spring Boot | HTML5 | Bootstrap |
| Java | CSS3 | JavaScript |
| Maven | Responsive Design | MVC Pattern |
